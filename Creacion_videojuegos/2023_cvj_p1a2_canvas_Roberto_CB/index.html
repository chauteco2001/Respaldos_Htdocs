Tutorial Usando Canvas

Qué es Canvas en Html

En HTML es un elemento que se utiliza para dibujar gráficos de forma programática en una página web. Proporciona un lienzo en blanco en el que puedes utilizar JavaScript para dibujar formas, imágenes y otros elementos gráficos, y animacion, tambien se utiliza para poder insertar figuras dentro del canvas deberemos de conocer cómo funciona el sistema de coordenadas.

Cómo llamar a canvas desde js

Para acceder al elemento canvas desde JavaScript, primero necesitas obtener una referencia a él utilizando el método getElementById o algún otro método de selección de elementos del DOM.
